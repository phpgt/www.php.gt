The legal stuff

PHP.GT is an open source project maintained by individuals rather than a commercial platform. The website exists to host documentation and information about the project, not to generate advertising revenue or monetise user data. Because the project is not built around user tracking or marketing analytics, there is no incentive to monitor visitors, profile users, or sell or share data with third parties.

What data is collected?

The website and all PHP.GT services do not collect any personally identifiable information from visitors or users.

This website uses a self-hosted analytics platform to understand which pages are viewed and how the site is used.

  • The analytics system does not use cookies.
  • It does not identify individual users.
  • It records only aggregated page view information.
  • Analytics data is retained for up to 8 years.

Like most websites, the web server automatically records technical information when a page is requested. This may include:

  • IP address
  • User agent (browser information)
  • Requested URL
  • Date and time of the request

These logs are used only for operating the website, diagnosing problems, and maintaining security.

Cookies

This website does not use cookies.

Data storage and sharing

This website does not perform cross-origin requests at all.

No cross-site tracking, behavioural profiling, or advertising tracking is taking place.

All data is stored on servers controlled by the website operator. No external analytics or data-processing services are used.
