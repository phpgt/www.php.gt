Page Not Found

This is a 404 error page. It has been served to your browser because the requested URI does not have a matching Page View within the project's page/ directory.

In case it wasn't obvious, this website is a WebEngine application. WebEngine allows the developer to create their own 404 pages by storing them at page/404.html. That's what you're looking at now.

When the Lifecycle class calls the Dispatcher::handle function, the Dispatcher attempts to create a ViewAssembly, which is a collection of streams that make up the whole response (such as the header, main page, and footer). If a matching main page can't be located to match the requested URI, the Assembly throws an exception which causes the handle function to be called again, but this time with a modified Request object - it's identical to the original request, apart from the URI is changed to /404 (or whatever error code is being served).
